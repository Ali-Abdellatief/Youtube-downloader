## About The Project

This Python script allows users to download YouTube videos in the highest resolution available. It utilizes the Pytube library to fetch video details and download the video.

<p align="right">(<a href="#readme-top">back to top</a>)</p>



<!-- INSTALLATION -->
## Installation

To run this script, you need to have Python installed on your system. Additionally, you need to install the Pytube library. You can install Pytube using pip:

```bash
pip install pytube
```

<p align="right">(<a href="#readme-top">back to top</a>)</p>



<!-- USAGE -->
## Usage

1. Run the script by executing the Python file in your terminal or command prompt.

```bash
python ytDownloader.py
```

2. Enter the YouTube video URL when prompted.

```bash
Enter the YouTube video you wish to download: [YouTube video URL]
```

3. The script will display the title and view count of the video.

4. The highest resolution version of the video will be downloaded to the specified location.

5. Once the download is complete, the script will display "Download Complete".
